Features

Everything you need to assess, track, and improve your observability posture.

🔍

Automated Assessment

Executes ~94 unique DQL queries (111 criteria with cross-entity ratios) against your Dynatrace tenant in real time via Grail.

📊

Dual Scoring

Coverage shows how much you cover. Maturity shows if you're covering in the right order, with progressive tier gating.

🎯

Interactive Radar Chart

Canvas-rendered polar chart with hover tooltips and click-to-drill-down into capability details and individual criteria.

🛠️

Guided Remediation

Specific actions and documentation links for every unmet criterion, organized by impact and difficulty.

📈

Evolution Over Time

Auto-saved snapshots to Dynatrace Document Store with A/B comparison of any two snapshots and delta analysis per capability.

📄

PDF Reports (EN/PT)

First Day Results report with dark-themed A4 design, TechRadar chart, capability breakdown, and remediation priorities. Available in English and Portuguese.

Scoring Model

Dual-dimension scoring with progressive maturity gating.

📊 Tier Weights (Progressive)

Foundation 60% weight Always counted
Best Practice 25% weight Requires Foundation ≥ 80%
Excellence 15% weight Requires Best Practice ≥ 60%

🏆 Maturity Levels

L0 Not Adopted Foundation < 50%
L1 Foundation Foundation ≥ 50%
L2 Operational F = 100% & BP ≥ 50%
L3 Optimized F+BP = 100% & E ≥ 50%

⚠ DPS Notice

Each assessment execution consumes DPS proportional to the GiB scanned in Grail. Actual cost depends on your tenant's data volume, Grail pricing tier, and contract terms.

Estimated cost: ~$0.92 – $1.42 per assessment (at $0.0065–$0.01/GiB). Running once per week: ~$4–6/month. Once per day: ~$28–43/month.

Entity queries (fetch dt.entity.*) and timeseries queries have zero or negligible cost. Identical queries are deduplicated and executed only once.

Quick Start

Get up and running in 5 steps.

1

Clone the repository

git clone https://github.com/mbdccoletta/Pulse-Assessment-App.git
2

Configure your tenant

Edit app.config.json — set environmentUrl to your Dynatrace tenant.

3

Install dependencies

npm install
4

Run locally (dev server)

npm run start
5

Deploy to production

npm run deploy
